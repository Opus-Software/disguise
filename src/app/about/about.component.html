About

Summary

Disguise is a project made with Angular that provides a user interface for the Mountebank server. It facilitates the user interactions with the API and helps visualize and manage all the informations of the imposters and stubs in the server. The source code for this application is located at https://github.com/Opus-Software/disguise.

Imposters Page

This is the page where you can manage the imposters present in the Mountebank server. You can visualize the informations of all the imposters and their stubs, add new imposters and delete existing ones. You can also add new singular stubs to imposters, edit and delete existing ones.
You can export the actual state of the Mountebank server, saving all the current imposters and stubs to a json file. It is also possible to import a configuration json file, adding the imposters and stubs defined in it to the running Mountebank server.
If an imposter has the attribute recordRequests set to true, you can visualize all its recorded requests. The requests are shown in a list with their respective informations like timestamp, headers and request body.

Logs Page

This is the page where you can visualize all the logs returned by the Mountebank server. The logs are presented in a table where you can filter them by the port number, the type of the log, the imposter related to the log and the log message.

Server Location

The application assumes by default that the Mountebank server is running on the port 2525 on localhost. If the user wishes to change this, it is necessary to modify the source code, changing the value of the target property in the src/proxy.conf.json file to the desired server location.
